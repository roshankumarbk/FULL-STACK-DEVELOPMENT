loop attribute is set to make the video play continuously without stopping.
muted attribute is not set, so the video will play with sound.
controls attribute is set to provide the user with playback controls such as play, pause, and volume.
width attribute is set to 400 pixels to define the width of the video player.
autoplay attribute is not set, so the video will not start playing automatically when the page loads.
src attribute specifies the source file of the video, which is "video1.mp4".
img is used for to link svgs or images
iframes is used to embed vedios from other platforms like youtube
